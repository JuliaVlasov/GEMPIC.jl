    Geometric ElectroMagnetic Particle-In-Cell Methods

    A Julia implementation of the GEMPIC

    This package implements a Finite Element Particle-in-Cell method based on the discretization of the underlying Hamiltonian structure of the Vlasov-Maxwell system.
